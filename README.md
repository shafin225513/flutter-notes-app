##  CRUD Logic Overview

The app implements full **CRUD (Create, Read, Update, Delete)** operations:

* Notes are created through a dedicated input flow
* Data is displayed in a structured list view
* Updates modify existing note entries
* Deletions instantly remove notes from both UI and storage

State updates are handled cleanly to ensure a responsive and predictable user experience.
